# Discord Bot

A Discord bot built for my personal server.
NOTE: It's still under initial development.

## Configuration

Create a `.env` file in the project's root directory with the following variables:

```text
TOKEN=your_bot_token
GUILD=your_server_id
```

* `TOKEN` — Your Discord bot token.
* `GUILD` — The ID of the Discord server where guild-specific slash commands will be registered.

## Features

* Basic moderation commands
* Ticketing system
* Additional utility features for server management
